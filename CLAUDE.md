# macbook-pro-dashboard

## Stack
React 19 + TypeScript + Vite + Tailwind CSS v4 + Recharts

## Status
Active

## Dev Commands
- `npm run dev` — Vite dev server
- `npm run build` — tsc -b && vite build
- `npm run lint` — ESLint

## Conventions
Follow [Vite conventions](~/.claude/conventions/web-vite.md) and [universal conventions](~/.claude/conventions/universal.md).
